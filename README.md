# TWMA Releases

TWMA 공개 베타의 공식 DMG와 체크섬을 제공하는 저장소입니다.

TWMA는 창을 배치하고, 작업공간으로 저장하고, 그대로 복원하는 macOS 메뉴 막대 앱입니다.

- 지원 환경: Apple Silicon Mac, macOS 14 이상
- 최신 베타: [TWMA 0.3.0 Beta 5](https://github.com/4celain/twma-releases/releases/tag/v0.3.0-beta.5)
- 버그 제보: [새 이슈 작성](https://github.com/4celain/twma-releases/issues/new?template=bug_report.yml)
- 개인정보: [개인정보 처리 안내](PRIVACY.md)
- 보안 제보: [보안 정책](SECURITY.md)

## 다운로드

Developer ID 서명, Apple 공증, Gatekeeper 검증을 마친 공개 베타만 [Releases](https://github.com/4celain/twma-releases/releases)에 게시합니다.

현재 공개 버전은 TWMA 0.3.0 Beta 5입니다.

이 저장소가 TWMA 공개 베타의 공식 배포·지원 채널입니다.
