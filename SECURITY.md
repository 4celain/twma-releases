# Security Policy

보안 취약점이나 개인정보 노출 가능성은 공개 이슈로 제보하지 마세요.

[GitHub의 비공개 취약점 제보](https://github.com/4celain/twma-releases/security/advisories/new)를 이용해 주세요. 원본 로그, 화면 캡처, 창 제목, 문서 내용, 계정 정보는 필요한 부분을 가린 뒤 최소한으로 첨부해 주세요.

일반적인 창 배치·작업공간 복원 문제는 [공개 버그 양식](https://github.com/4celain/twma-releases/issues/new?template=bug_report.yml)을 이용할 수 있습니다.

---

Please do not report security vulnerabilities or potential privacy exposures in a public issue.

Use [GitHub private vulnerability reporting](https://github.com/4celain/twma-releases/security/advisories/new). Redact and minimize any logs, screenshots, window titles, document contents, or account information before attaching them.
