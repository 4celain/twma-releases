# TWMA Releases

TWMA 공개 베타의 공식 DMG와 체크섬을 제공하는 저장소입니다.

TWMA는 창을 배치하고, 작업공간으로 저장하고, 그대로 복원하는 macOS 메뉴 막대 앱입니다.

- 지원 환경: Apple Silicon Mac, macOS 14 이상
- 제품 사이트: [TWMA](https://twma-for-mac.mamasgun.chatgpt.site)
- 버그 제보: [새 이슈 작성](https://github.com/4celain/twma-releases/issues/new?template=bug_report.yml)

## 다운로드

Developer ID 서명, Apple 공증, Gatekeeper 검증을 마친 공개 베타 DMG만 [Releases](https://github.com/4celain/twma-releases/releases)에 게시합니다.

첫 공개 베타를 준비 중입니다. 검증 전 개발 빌드는 이 저장소에 올리지 않습니다.
