# TWMA Releases

TWMA 공개 베타의 공식 DMG를 제공하는 저장소입니다.

TWMA는 창을 배치하고, 작업공간으로 저장하고, 그대로 복원하는 macOS 메뉴 막대 앱입니다.

- 지원 환경: Apple Silicon Mac, macOS 14 이상
- 최신 베타: [TWMA 0.3.0 Beta 6 (Build 8)](https://github.com/4celain/twma-releases/releases/tag/v0.3.0-beta.6)
- 다운로드: [TWMA-0.3.0-beta.6-build.8-arm64.dmg](https://github.com/4celain/twma-releases/releases/download/v0.3.0-beta.6/TWMA-0.3.0-beta.6-build.8-arm64.dmg)

## 다운로드

Developer ID 서명, Apple 공증, Gatekeeper 검증을 마친 공개 베타만 [Releases](https://github.com/4celain/twma-releases/releases)에 게시합니다.

현재 공개 버전은 TWMA 0.3.0 Beta 6 (Build 8)입니다. 다운로드 검증용 SHA-256은 [릴리스 본문](https://github.com/4celain/twma-releases/releases/tag/v0.3.0-beta.6)에서 확인할 수 있습니다.

이 저장소가 TWMA 공개 베타의 공식 배포·지원 채널입니다.
